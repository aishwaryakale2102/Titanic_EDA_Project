# Titanic EDA Project

## Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns, relationships, missing values, and survival trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
1. Loaded the Titanic dataset
2. Checked dataset information and summary statistics
3. Identified missing values
4. Filled missing Age and Embarked values
5. Dropped Cabin column
6. Performed univariate analysis
7. Performed bivariate analysis
8. Performed multivariate analysis
9. Created pairplot and heatmap
10. Summarized findings

## Key Findings
- Most passengers were between 20 and 40 years old
- Female passengers had higher survival rates
- First-class passengers survived more than third-class passengers
- Fare contains several outliers
- Higher fare passengers had better survival chances
- Passenger class and fare are strongly related
